Spring Boot 实践笔记

2017-04-27

介绍

  • SpringBoot是SpringMVC的升级版，两者没有直接的联系。
  • 特点
    • 简化配置，抛掉繁琐的xml配置文件。
    • 下一代框架，受关注度高。
    • 微服务的入门级微框架。
      • 微服务：当下流行的一种架构/解决方案
      • SpringCloud：是Spring家族为微服务提供的一整套服务
      • SpringCloud 基于SpringBoot开发，学习SpringCloud必须先学习SpringBoot

Controller使用

  • @Controller :处理http请求

  • @RestController:处理请求并声明返回数据

    • 等同于@Controller+@ResponseBody

  • @RequestMapping:配置url映射

    • value ：url映射地址。可以为数组,格式为

    • method：定义请求方式

      • 不指定条件下，任何类型的请求都可以成功

  • @PathVariable:获取url中的数据

  • @RequestParam：获取请求参数的值

    • 参数
      • value:参数名
      • required：是否必填
      • defaultValue：默认值，必须是字符
    • 示例

  • @GetMapping：组合注解

    • 等同于@RequestMapping + method = RequestMethod.GET

  • @PostMapping:组合注解

    • 等同于@RequestMapping + method = RequestMethod.POST
